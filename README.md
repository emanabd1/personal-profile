 Eman Abdulsemed | Dynamic Personal Portfolio & CV

## Overview
This repository hosts a modern, responsive, and interactive personal portfolio website for Eman Abdulsemed, a 2nd-year Electrical Engineering and Computing student at Adama Science and Technology University with a strong passion for development. The portfolio serves as a central hub to showcase skills, projects, and provide essential contact information, along with a dedicated CV page.

Designed with a focus on clean aesthetics, user experience, and robust front-end implementation, it aims to make a memorable impression.

## Features
•   **Vibrant & Modern UI/UX:** An attractive and professional design incorporating a pleasing color palette, gradients, and custom typography (`Montserrat` and `Poppins` fonts).
•   **Responsive Layout:** Fully optimized for seamless viewing across all devices – desktops, tablets, and mobile phones.
•   **Interactive Bio Section:** Introduces Eman Abdulsemed with a compelling description of her interests, studies, and career aspirations, featuring subtle hover animations on the profile image.
•   **Engaging Call-to-Action:** Prominent buttons to "Get in Touch" and "View My CV" to guide visitors.
•   **Intuitive Contact Form:**
    •   Client-side validation using JavaScript ensures required fields are filled and email format is correct.
    •   A professional, animated success modal provides feedback upon successful (simulated) submission, enhancing user experience beyond standard browser alerts.
•   **Dedicated CV Page:** A separate, well-structured, and print-friendly CV (Curriculum Vitae) page for a detailed overview of education, skills, and projects.
•   **Clean Codebase:** Organized HTML structure, modular CSS with variables for easy customization, and clear JavaScript for interactivity.
•   **Smooth Animations:** Subtle CSS animations and transitions (e.g., fade-in sections, button hovers) add a polished touch.

## Technologies Used
•   **HTML5:** For semantic structure and content.
•   **CSS3:** For styling, layout (Flexbox, Media Queries for responsiveness), custom properties (variables), and animations.
•   **JavaScript (ES6+):** For client-side form validation and modal interactivity.
•   **Google Fonts:** `Montserrat` (for headings) and `Poppins` (for body text) for enhanced typography.
